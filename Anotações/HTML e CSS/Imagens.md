# Imagens

| TAG                                | FUNÇÃO                                    |
| ---------------------------------- | ----------------------------------------- |
| img>                               | Tag HTML                                  |
| img> src="caminho"                 | Caminho da imagem é obrigatório           |
| img> src="caminho" alt="descrição" | Descrição para a imagem, não obrigatório. |
