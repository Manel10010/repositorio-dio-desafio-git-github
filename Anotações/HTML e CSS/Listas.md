# Listas

| TAG  | FUNÇÃO                                                  |
| ---- | ------------------------------------------------------- |
| ul>  | Tag HTML para Listas onde a ordem dos itens não importa |
| ol>  | Tag HTML para Listas onde a ordem dos itens importa     |
| li>  | Item da lista                                           |
